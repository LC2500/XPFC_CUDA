# XPFC_CUDA
The following CUDA/C++ program is based on the structural phase field crystal method (XPFC), PRL 105, 045702 (2010) and PRE 83, 031601 (2011). The code makes use of cuFFT and NCCL from the NVIDIA HPC SDK by NVIDIA. 


## Requirements

This program requires the NVIDIA CUDA Toolkit (or CUDA HPC SDK) to compile and run.
It can be downloaded from the official NVIDIA website:

- [NVIDIA CUDA Toolkit](https://developer.nvidia.com/cuda-downloads)
- [NVIDIA HPC SDK](https://developer.nvidia.com/hpc-sdk)
