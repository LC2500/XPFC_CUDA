# XPFC_CUDA
The following CUDA/C++ program is based on the structural phase field crystal method (XPFC), PRL 105, 045702 (2010) and PRE 83, 031601 (2011). The code makes use of cuFFT and NCCL from the NVIDIA HPC SDK by NVIDIA. The NVIDIA HPC SDK should be downloaded separately by the user at https://developer.nvidia.com/hpc-sdk.
